Enables Sepia for Imagick in WordPress

== Description ==

This plugin adds the sepia() function to to the Imagick editor within WordPress.

== Installation ==

1. Upload the folder `imagick-sepia` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
